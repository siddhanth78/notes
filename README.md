# Notes

### Simple PyGame-based notes app
- barebones no bloat
- 60fps for smoother experience
- vim-like controls

### Terminal-based file navigation
- Create, delete, rename files and dirs

Supports any non-binary filetype

### Controls: Regular mode
- Arrow keys: Move cursor
- Esc: Exit app
- Ctrl-S: Save file
- Ctrl-O: Insert line above current line
- Ctrl-Shift+O: Insert line below current line
- Ctrl-D: Delete current line
- Ctrl-B: Move to beginning of current line
- Ctrl-E: Move to end of current line
- Ctrl-F: Find
- Ctrl-N: Go to next match
- Ctrl-R: Find and replace
- Ctrl-V: Vision mode (Begins at cursor)
- Ctrl-P: Paste clipboard
- Ctrl-Shift-"+": Zoom in
- Ctrl-Shift-"-": Zoom out
- Enter: New line

### Controls: Vision mode
- Arrow keys: Modify selection
- C: Copy selection to clipboard
- X: Cut selection to clipboard
- P: Replace selection from clipboard
